# Tainted Grail - Quests Tracker
[![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white)](https://en.cppreference.com/w/cpp/language)
[![Qt](https://img.shields.io/badge/Qt-%23217346.svg?style=for-the-badge&logo=Qt&logoColor=white)](https://doc.qt.io/)
[![Android](https://img.shields.io/badge/Android-%a4c639.svg?style=for-the-badge&logo=android&logoColor=white)](https://www.android.com/)

While playing the board game [Tainted Grail](https://awakenrealms.com/games/awaken-realms/tainted-grail), I found difficult to track side quests. The game leaves you with a small area in a sheet to keep track of every important elements of the game (and there are many !). This app aims to help tracking relevant information during a playthrough, with a small map to help visualizing.

## Requirements
Using Qt Maintenance tools you need to install
* Qt 6.8.13 (With Android / a Windows compiler depending on your needs)
* Qt Creator

This app has been designed for Android. The Windows compiler only help for faster development. To configure QtCreator for Android, please follow [this link](https://doc.qt.io/qt-6/android-getting-started.html).
